Install firmware

Connect the PiLab Panel ESP32-S3 board over USB. Use a Chromium-based desktop browser such as Chrome, Edge, or Brave. Click the install button and select the board serial port.

Choose the firmware that matches your board flash size. If the serial monitor shows Detected size(8192k) smaller than ... header(16384k), use the 8MB flash version.

Before flashing

  1. Use a USB data cable, not a charge-only cable.
  2. Close idf.py monitor, serial terminals, or other programs using the board.
  3. If auto-reset fails, hold BOOT, tap RESET, then release BOOT.
  4. For a first install, allow erase when prompted.

After flashing

After boot, connect to the panel access point, then open http://192.168.50.1. Once the panel joins your local WiFi network, you can also use mDNS at http://pilab-panel.local when available.

Maintainer notes

  • Run tools\copyFirmware.ps1 after idf.py build to copy binaries into webflasher\firmware\.
  • The copy script regenerates manifest.json from ESP-IDF build metadata when build\flasher_args.json is available.
  • Publish the webflasher folder with GitHub Pages or serve it locally for testing.
