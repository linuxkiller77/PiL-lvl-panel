# PiLab Panel Factory Demo

This folder contains the same demo content embedded in the firmware on a fresh flash.

Files:

- `hmi_layout.json` - multi-screen HMI layout for the LCD runtime.
- `hmi_script.as` - AngelScript event handlers used by the buttons, toggle, and setpoints.
- `hmi_tags.json` - starter tag registry for the demo.

Use the browser UI to import or edit these files, then use **Save to RAM** for quick testing and **Save to Flash** when you want the panel to boot with the changes.
